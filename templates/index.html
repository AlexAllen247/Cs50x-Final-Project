Cs50x has been a fantastic learning experience. It has broadened my knowledge of computer programming and computer science. I only began my computer programming journey 5 months ago and I feel I have learned so much already, with still a lot more to go. So, I decided to use the final project of Cs50x as a homage and portfolio of sorts to display the things I have learned during the course.

Although this may not meet all of the requirements of a Final Project. This is not going to change the world, it doesn't solve any major current issues and it is unlikely going to impact the community. However, in order to achieve these goals everyone must start from humble beginnings. I may not be ready to change the world, however I have begun my journey to one day impact it.

Each of the projects here represent some part of the Cs50x course. Balls is a Javascript version of my original Scratch project from week 0, which can be found here.
